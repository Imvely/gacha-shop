## 결정 사항
- rng는 CSPRNG가 아니라 seed에서 유도(HMAC) — 회차 종료 후 제3자 재현 검증이 목적
- 서버는 오픈 시점부터 seed 원본 보관(rng 유도에 필요), API 노출은 완판 후에만
- 확률 표시는 최대잉여법(0.01% 단위) — 반올림 후에도 합계 정확히 100.00% 보장
- 인증은 F-01 전까지 X-User-Id 헤더 스텁 (app/deps.py, 프로덕션 전 제거 필수)
